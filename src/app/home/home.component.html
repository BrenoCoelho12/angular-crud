  • Este é um CRUD desenvolvido am Angular + Typescript. Os dados são persistidos no Firebase Realtime Database, um banco de dados na nuvem NoSQL. O front-end foi construído utilizando bootstrap 4.
  • O contexto do CRUD é o de simular o cadastro, atualização, deleção e listagem de produtos de uma loja de produtos mecânicos.
